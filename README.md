# Définition 

La définition même du Dark Tourism prendra un grosse place dans notre projet. L'idée est d'essayer de poser la limite de ce qu'est le Dark Tourirm. Est ce que c'est les Safaris humains, la visite de Tchernobyl ou la visite du mémorial 9/11? 
L'objectif est de se faire questioner les visiteur.ices de notre site. 

# Top 10

Le deuxième aspect est du projet serait de proposer un Top 10 des lieux de dark tourism les plus vissités en Europe ou dans le Monde. Le but est la aussi de faire questionner les visiteur.ices avec des exemples concrets.
